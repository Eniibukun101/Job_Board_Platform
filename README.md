# Job_Board_Platform
A full-stack job board built with Node.js And React

## Setup 

### Backend
cd into bsackend folder
Run 'npm install'
Copy '.env.example' to '.env' and fill in values
Run 'npm start'

### Frontend
cd into frontend folder
Run 'npm install'
Run 'npm start'

### Enviroment
See '.env.example'
